# btn
Button component for vue

## Usage

### Install with npm:
```
npm install https://github.com/goudev/btn.git --save
```

### Install with yarn:
```
yarn add https://github.com/goudev/btn.git
```

### How to use:
```javascript
<script>
import btn from 'btn'

export default {
  components: {
    btn
  }
}
</script>
```
```html
<template>
  <btn>My Button</btn>
</template>
```
